# Thala Footwear - Sneaker Store

A clean, modern, static e-commerce website for **Thala Footwear** (Thala7), inspired by MS Dhoni's "Thala" legacy. Built with pure HTML and CSS — perfect for a portfolio project or a quick landing page.

![Website Preview](https://abrosshoes.com/cdn/shop/files/Cliff_0008_000A7864.jpg?v=1756296498&width=600)

## ✨ Features

- **Fully Responsive Design** – Works great on mobile, tablet, and desktop
- **Modern Dark Theme** with premium red accents (#e74c3c)
- **Hero Section** with subtle background animations
- **Product Detail Page** with image gallery (thumbnails + hover effects)
- **Shopping Cart** page with item management UI
- **Checkout** page with payment method selection (static)
- **Sticky Navbar** with cart counter and profile avatar
- **Font Awesome** icons throughout
- **Clean, maintainable CSS** with good use of Flexbox and Grid

## 📁 Project Structure
Thala-Footwear/

text## 🛠️ Technologies Used

- **HTML5** – Semantic markup
- **CSS3** – Flexbox, Grid, animations, custom properties
- **Font Awesome 6** – Icons (CDN)
- No JavaScript (static demo)

## 🎨 Design Highlights

- Dark modern aesthetic (`#0f172a` background)
- Bold red accent color matching the "Thala" branding
- Smooth hover transitions and micro-interactions
- Mobile-first responsive breakpoints
- Sticky navigation and order summary

## 🚀 How to Run

1. Clone or download the project
2. Open any `.html` file in your browser (no build step needed)
3. Start with `index.html`
